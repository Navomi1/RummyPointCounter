# Rummy Point Counter

An iOS app for tracking points in the card game Rummy.

## Features

- Add as many players as you want to the game: new players start with the highest current score among existing players (or 0 if they're the first player)
- View all players with their total scores (sum of all rounds)
- View all previous rounds showing by individual round scores

## How to Use

1. **Add Players**: Tap the "+" button to add players. The first player starts at 0 points, subsequent players start with the highest current score.

2. **Add Round Scores**: Tap "Add New Round" to enter scores for all players for a new round.

3. **View History**: Tap "View Round History" to see all previous rounds with individual scores.

4. **Track Totals**: The main screen shows each player's total score (sum of all their rounds).

## Building the App

1. Open Xcode
2. Create a new iOS App project named "RummyPointCounter"
3. Replace the generated files with the files in this repository
4. Build and run on iOS simulator or device (iOS 15.0+)

## Requirements

- iOS 15.0 or later
- Xcode 13.0 or later
- SwiftUI
